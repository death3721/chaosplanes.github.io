Markdown常用语法

发表于 2017-06-25

标题

以此类推, 总共六级标题, 注意需要在井号后加入一个空格, 这是标准的Markdown语法.

列表

无序列表

列表的显示只需要在文字前加上 - 或 * 即可变为无序列表, 注意需要加一个空格.

有序列表

有序列表直接在文字前加1. 2. 3. 符号, 要和文字之间加上一个空格。

引用

如果需要引用一小段别处的句子, 只需要在文本前加入 >(大于号) 即可.

图片与链接

链接为：baidu

粗体与斜体

Markdown 的粗体和斜体也非常简单，用两个 包含一段文本就是粗体的语法，用一个 包含一段文本就是斜体的语法。
