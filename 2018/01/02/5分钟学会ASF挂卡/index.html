5分钟学会ASF挂卡

发表于 2018-01-02

前言

由于近期steam社区域名被污染, 导致idle master不能挂卡, 于是转战ASF, 但是论坛置顶的帖子很容易让人望而生畏.

本文仅面向想单纯挂卡的玩家.

准备工作

1.更新系统运行库

win32

下载并安装 VC++ Redistributable: 下载地址

下载并安装 .NET Core 2.0 SDK: 下载地址

win64

下载并安装 VC++ Redistributable: 下载地址

2.下载最新的ASF:

ASF官方下载: 下载地址

自行选择系统版本, 本文仅以win32/win64为例. 32位系统选择下载Generic版即可.

3.设置配置文件

ASF V3网页版配置文件: 网页地址

设置好后, 点击下方download下载即可.

4.置入配置文件

把刚刚下载完的配置文件ASFtest.json放入第一步下载的ASF文件夹根目录下的config文件夹下即可.

5.创建bat文件

创建bat文件的目的是方便调用命令行工具实现一键开启挂卡, 否则需要每次打开命令行工具.
